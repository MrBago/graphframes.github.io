Attach DataFrame to R search path

Description

The specified DataFrame is attached to the R search path. This means that the DataFrame is searched by R when evaluating a variable, so columns in the DataFrame can be accessed by simply giving their names.

Usage

attach(what, pos = 2L, name = deparse(substitute(what)),
  warn.conflicts = TRUE)

## S4 method for signature 'DataFrame'
attach(what, pos = 2,
  name = deparse(substitute(what)), warn.conflicts = TRUE)

Arguments

what

(DataFrame) The DataFrame to attach

pos

(integer) Specify position in search() where to attach.

name

(character) Name to use for the attached DataFrame. Names starting with package: are reserved for library.

warn.conflicts

(logical) If TRUE, warnings are printed about conflicts from attaching the database, unless that DataFrame contains an object
