as.data.frame {SparkR}R Documentation

Download data from a DataFrame into a data.frame

Description

This function downloads the contents of a DataFrame into an R's data.frame. Since data.frames are held in memory, ensure that you have enough memory in your system to accommodate the contents.

Usage
